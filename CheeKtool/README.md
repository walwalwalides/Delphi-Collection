# CheeKtool
- Special Tools Used For Specific Tasks .               
![](Form/CheecKtool.png) 


# Features  

- Opening Text Log Files.
- Delete all temporary files.
- Load all dynamics dll in a directory and check the dependency.
- Dynamically load and call a function in a BPL package.
